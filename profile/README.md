# TNO GenAI Lab
In 2026 the TNO GenAI Lab is set-up: a centralized landing page for code and packages. TNO,Nederlandse Organisatie voor toegepast-natuurwetenschappelijk onderzoek, is a research institute in the Netherlands. This page is dedicated to promote code transfer, collaboration and reusability of project results related to Generative AI.
This includes topics such as  Agentic AI, Hybrid AI, LLM Evaluation, Synthetic Data Generation, Explainable AI and RAGs.
The coming period will be used to further expand and fill this Opensource page.

Feel free to reach out ot us with questions, feedback and suggestions!
